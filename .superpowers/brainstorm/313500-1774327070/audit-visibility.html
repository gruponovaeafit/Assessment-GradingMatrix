Audit Log Visibility

How should administrators view these logs?

A

Database Access Only (MVP)

Logs are only viewable by developers with direct access to the Supabase dashboard or SQL editor. Quickest to implement.

B

Super-Admin Dashboard

Add a dedicated "Audit Logs" tab in the Super-Admin view to see all system-wide actions with filtering and search.

C

Scoped Admin View

Group Admins can see logs related only to their own assessments and staff. Most secure and decentralized.
